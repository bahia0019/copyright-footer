== Description ==
This is a Full Site Editing Footer block.

Adds a footer area with a Copyright date that updates every year automatically, with some additional options.

To be used as a block in the Footer Template Part with full site editing.

== Installation ==

1. Install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress


== Frequently Asked Questions ==

= Can I use this as a regular block in content? =

No, this is meant to be used as a Template Part in themes that support full site editing.

== Screenshots ==

1. Use this block in the Footer Template Part.
2. Copyright year, and Site Name are enabled by default.
3. You can optionally enable a text area that allows you to display legal links like Privacy Policy or Terms of Use pages.
4. You can modify the colors of Text, Links and Background.
